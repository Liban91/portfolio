A) HTML Structure - top

This template is a single page responsive design using Bootsrap. Mutant Portfolio Template has three variations:

  • Default Version : Single image background in home section.
  • Slideshow Version : Background image slideshow in home section.
  • Particle Version : Animated particle overlay in the background in home section.
You will find the main content of this template in "mutant" folder:

Open index-default.html to access the default version.
Open index-slideshow.html to access the slideshow version.
Open index-particle.html to access the particle version.

Upload the whole contents of the mutant folder to the web hosting server and keep the desired version's index file and rename it to index.html and delete the other two versions index file. For example, if you want to use the default version, rename the index-default.index file to index.html. Then delete the index files of other two versions, index-slideshow.html and index-particle.html .

All of the information within the main content area is nested within a div with a class of "all-container". All the sections are nested within this "all-container" classed div with unique class names(i.e header,menu,about-me,services,portfolio,testimonials,contact). Do not edit the class names, if you have to, make sure that you change the respective class name in the main.css, responsive.css and main.js files. Basic HTML structure looks like this:

<!DOCTYPE html>
<html lang="en" class="no-js">

To change the website title,favicon and meta-data, you would head over to the head tag and edit the content within the respective tag:

Change website title here: 
<title> Mutant Portfolio Template </title>

Change website favicon here: 
<link rel="icon" type="image/png" href="image/favicon/favicon.png">

There are 2 version of logo used in this template - Light version for dark backgrond and Dark version for light background. You can add your logo or delete them here: 

Use light version of logo here: 
<div class="header section" id="1"> 
  <div class="logo">
    <img src="image/logo/logo.png" alt="Logo">
  </div>
</div>

Use dark version of logo here: 
<div class="menu" id="menu">
 <div class="mobile-menu-logo"><a href="#"><img src="image/logo/logo-dark.png" alt="Logo"></a></div>
</div>

You can use a single logo for both the version if you want.

To change the animated text, find "cd-headline" classed element in the header section. Then change the content within the <b> tag. There are total two animated texts in this template.

Change the content of animated text here:

 </strong>
</div>

B) CSS Files and Structure - top

I'm using two CSS files dedicated to style this template. You can find all the CSS files in the "css" folder withing the "mutant" folder.

  • main.css : To style the template.
  • responsive.css : To style the template in small screen devices
Main CSS files of this template: 
<link rel="stylesheet" type="text/css" href="css/main.css"/>
<link rel="stylesheet" type="text/css" href="css/responsive.css">

main.css and resonsive.css are all the files you will be needing to change the design of this template. This two files have the same structure: 

Structure of main.css and responsive.css: 
/*--------------------------------
   1. Start of a section
----------------------------------*/
body{
	style:value;
}
/*--------------------------------
   1. End of a section
----------------------------------*/

CSS files are indexed into sections. You can find the index lebel at the top of the file.If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Background image of header section, services section and contact section can be changed in the main.css file by scrolling in the respective section.
